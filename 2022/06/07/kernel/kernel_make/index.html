编译

  • 屏蔽编译信息

  • 使用make -j<n>参数加速编译

  • 指定编译某些模块

  • 使用 verbose 模式，将每一步执行的命令都打印出来，并重定向到一个文件中去，这样以后可以方便地查找模块之间的依赖关系。
  • 使用 ccache 提高编译速度，使用 ccache时，需要更改源码树根目录下面的 Makefile 文件，在 CC 和 HOSTCC 变量的定义前添加 ccache。ccache主页。
