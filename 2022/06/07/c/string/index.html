strcpy

  • 源指针所指的字符串内容是不能修改的，因此应该声明为 const 类型。
  • 要判断源指针和目的指针为空的情况，这里使用assert。
  • 要用一个临时变量保存目的串的首地址，最后返回这个首地址。
  • 函数返回 char* 的目的是为了支持链式表达式，即strcpy可以作为其他函数的实参。

  • 若s1>s2，返回正数；
