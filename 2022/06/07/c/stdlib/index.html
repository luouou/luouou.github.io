c standard library

发表于 2022-06-07 更新于 2022-06-08 分类于 c语言 阅读次数：

c语言标准函数库分为以下几类：

  • 输入/输出 I/O
  • 字符串和字符处理
  • 数学
  • 时间、日期和本地化
  • 动态分配
  • 其他
  • 宽字符函数

assert.h

assert.h头文件提供了一个 assert 的宏，用于在函数开始处检验传入参数的合法性。assert会先判断表达式的真假，如果表达式值为假，那么它先向stderr打印错误信息，然后通过调用 abort 来终止程序运行。如果表达式值为真，继续运行后面的程序。

assert只在 DEBUG 下生效，在release版本中，可以通过在#include <assert.h>语句之前插入#define NDEBUG来禁用assert调用。

ctype.h

ctype.h 头文件提供了一些用于测试和映射字符，大小写转换函数。

errno.h

errno.h 头文件定义了整数变量 errno，就近发生的系统调用错误都会存储到 errno中，下一次的错误码会覆盖掉上一次的错误。可使用errno命令查阅错误码定义。

float.h

float.h 头文件包含了一组与浮点值相关的依赖于平台的常量。

limits.h

limits.h 定义了各种变量类型的最大值宏。

time.h

time.h 头文件定义了各种操作日期和时间的函数。

stdlib.h

stdlib .h 头文件定义了各种通用工具函数。

EXIT_FAILURE这是 exit 函数失败时要返回的值。
EXIT_SUCCESS这是 exit 函数成功时要返回的值。
RAND_MAX这个宏是 rand 函数返回的最大值。
MB_CUR_MAX 这个宏表示在多字节字符集中的最大字符数，不能大于 MB_LEN_MAX。

string.h

string .h 头文件定义了各种操作字符数组的函数。

math.h

math.h 头文件定义了各种数学函数。

stdarg.h

stdarg.h 头文件定义了一个变量类型 va_list 和获取可变参数中的参数的方法。

stddef.h

stddef .h 头文件定义了各种变量类型和宏。

ptrdiff_t 这是有符号整数类型，它是两个指针相减的结果。
size_t 这是无符号整数类型，它是 sizeof 关键字的结果。
wchar_t 这是一个宽字符常量大小的整数类型。

signal.h

signal.h 头文件定义了一些跟信号相关的函数和宏。

stdio.h

stdio .h 头文件定义了各种函数来执行输入和输出。
